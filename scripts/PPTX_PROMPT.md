# PROMPT: IPS External Strategic Analysis — PowerPoint Deck

Create a polished, EMBA-quality 16:9 PowerPoint presentation (11 slides) using PptxGenJS. This is for a Xavier EMBA Capstone presentation to IPS leadership. It needs to look like it came from McKinsey — clean, data-dense, executive-grade.

## Design Direction

- **Color palette:** Primary `#4950BC` (IPS purple), secondary `#1A1A2E` (navy), accent `#E8E9F7` (light purple), dark `#1A1A1F`, white `#FFFFFF`, off-white `#F5F5F8`
- **Data colors:** Red `#DC2626` (threats/gaps), Green `#166534` (tailwinds), Amber `#92400E` (headwinds), Teal `#0D9488`, Orange `#D97706`
- **Fonts:** Georgia for titles (36-44pt), Calibri for body (14-16pt), Calibri for captions (7-10pt)
- **Every slide has:** Purple header bar, act label, source citations at bottom
- **Visual motif:** Numbered circles for findings, stat callout cards with big numbers, color-coded category badges
- **NO bullet-only slides.** Every slide needs shapes, tables, stat cards, or visual elements.
- **NO accent lines under titles.** Use whitespace and background color blocks instead.
